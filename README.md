# HatsProtocolSample

HatsProtocol を調査・学習するためのサンプルリポジトリです。

## 動かし方

- インストール

  ```bash
  yarn
  ```

- 環境変数の設定

  `pkgs/sample` ディレクトリ直下に `.env`ファイルを作成し、以下の変数をセットする。

  ```txt
  PRIVATE_KEY=""
  RPC_URL=""
  ```

  `pkgs/sample-hats-module`ディレクトリ直下に `.env`ファイルを作成し、以下の変数をセットする。

  ```txt
  GC_RPC=
  POLYGON_RPC=
  INFURA_KEY=
  export PRIVATE_KEY=
  ETHERSCAN_KEY=
  GNOSISSCAN_KEY=
  POLYGONSCAN_KEY=
  OPTIMISM_KEY=
  ARBISCAN_KEY=
  FOUNDRY_PROFILE=
  ```

- サンプルスクリプトの実行

  ```bash
  yarn sample sample
  ```

- HatID を IP に変換する

  ```bash
  yarn sample hatIdToIp
  ```

  ```bash
  hatIdHex: 0x000001c000020000000000000000000000000000000000000000000000000000

  ```
